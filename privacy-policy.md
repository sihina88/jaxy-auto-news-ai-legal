## Privacy Policy
**Jaxy Auto News AI**

Last updated: March 2026

Jaxy Auto News AI is an automated application that collects publicly available news from RSS feeds and posts summarized content to a Facebook Page.

This Privacy Policy explains how the application handles data.

---

### 1. Information We Collect

Jaxy Auto News AI does **not collect, store, or process personal user data**.

The application only processes:

- Public news articles from RSS feeds
- AI-generated summaries
- Content posted to a Facebook Page

No personal user information such as names, emails, or private data is collected.

---

### 2. How the Application Works

The application performs the following automated tasks:

1. Fetches news articles from publicly available RSS feeds
2. Uses AI to generate summaries and captions
3. Allows an administrator to approve posts
4. Publishes approved posts to a Facebook Page

The application does not interact with Facebook users or collect any Facebook user information.

---

### 3. Third-Party Services

This application may use third-party services including:

- Meta / Facebook Graph API
- OpenAI API
- Firebase (Google Cloud)

These services may process limited data necessary for application functionality.

Please refer to their respective privacy policies.

---

### 4. Data Storage

The system stores only technical data necessary for operation, such as:

- News article metadata
- Generated captions
- Post status information

No personal user data is stored.

---

### 5. Data Security

Reasonable technical measures are used to protect system data and API credentials.

---

### 6. Data Deletion

Since the application does not collect personal user data, there is no personal data to delete.

If you have any concerns about content posted by the application, please contact the administrator.
